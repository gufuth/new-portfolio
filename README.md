# new-portfolio

Ian Luna's portfolio rebuild. Cinematic noir, Paris Texas palette, Last Stop Diner concept.

## Stack
- Astro 5 with React islands + MDX
- Tailwind v4 (via @tailwindcss/vite)
- Fraunces (display) + Lexend (body) + Geist Mono (script blocks), self-hosted
- Motion + Lenis + GSAP (animation)
- View Transitions API (native page transitions)

## Local dev
```
npm install
npm run dev
```

## Build
```
npm run build
```

## Deploy
Cloudflare Pages auto-deploys on push to `main`.
